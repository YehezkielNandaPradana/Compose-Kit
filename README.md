# 📦 Compose Kit

Compose Kit is a modern Android Inventory Management application built using **Kotlin** and **Jetpack Compose**. The project showcases modern Android development practices including Material 3, MVVM architecture, reusable UI components, and reactive state management.

Designed as a portfolio project, Compose Kit demonstrates how to build clean, scalable, and maintainable Android applications using Google's latest development tools.

---

## ✨ Features

- 📋 Inventory Item Management
- ➕ Add New Items
- 🔍 Real-time Search
- 🏷️ Category Filtering
- 🎨 Modern Material 3 UI
- 📱 Fully Responsive Compose Layout
- ♻️ Reusable UI Components
- ⚡ Reactive State Management
- 🧩 Modular Project Structure
- 🌙 Clean and Minimal Design

## 🏛️ Architecture

Compose Kit follows the **MVVM (Model-View-ViewModel)** architecture.

```
UI (Compose)
      │
      ▼
 ViewModel
      │
      ▼
  Model/Data
```

This architecture provides:

- Better Separation of Concerns
- Easier Testing
- Improved Maintainability
- Reactive UI Updates
